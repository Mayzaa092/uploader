API Integration

Automate your uploads using our simple REST API.

POST ENDPOINT
https://uploader.mayzaa.my.id/api/upload
CURL EXAMPLE
curl -X POST https://uploader.mayzaa.my.id/api/upload \ -F "files=@yourfile.png"
RESPONSE FORMAT
{ "author": "Mayzaa", "file": "yourfile.png", "link": "https://uploader.mayzaa.my.id/api/upload/yourfile.png" }
SHORTEN ENDPOINT (GET /api/shorten?url=...)
curl "https://uploader.mayzaa.my.id/api/shorten?url=https%3A%2F%2Fimage.pollinations.ai%2Fprompt%2Fblackhole%3Fmodel%3Dflux%26width%3D1024"
{ "author": "Mayzaa", "code": "aZ3kq1", "original": "https://image.pollinations.ai/prompt/...", "link": "https://uploader.mayzaa.my.id/s/aZ3kq1" }
CLONE ENDPOINT (POST /api/clone, maks 25 MB)
curl -X POST https://uploader.mayzaa.my.id/api/clone \ -H "Content-Type: application/json" \ -d '{"url":"https://contoh.com/file.png"}'
{ "author": "Mayzaa", "file": "file.png", "link": "https://uploader.mayzaa.my.id/api/upload/file.png", "size": 123456, "clonedFrom": "https://contoh.com/file.png" }
